# PrismBI

**PrismBI** — концепция корпоративной BI-платформы на базе приложения Prism: единое место для витрин данных, дашбордов и управленческой отчётности.

## Зачем это нужно

- **Сводка из разных источников.** Подключайте базы, файлы и API — получайте согласованные метрики без «ручных» Excel-сводок.
- **Дашборды и KPI.** Наглядные панели для руководства и операционных команд: цели, отклонения, drill-down к деталям.
- **Доступ и безопасность.** Роли, аудит и разграничение чувствительных показателей между подразделениями.
- **Интеграции.** REST и выгрузки для встраивания в порталы, почтовые рассылки отчётов и обмен с учётными системами.

## Технический стек (репозиторий)

Ruby on Rails, Tailwind CSS, Hotwire/Turbo по мере развития проекта. Лендинг в репозитории отражает продуктовую подачу PrismBI как BI-системы.

## Запуск

Стандартные шаги для Rails-приложения: установка зависимостей, настройка базы данных, `bin/rails server`. Подробности см. в официальной документации Rails для вашей версии.
